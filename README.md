# Paytm-clone
A fully responsive PayTM clone built using only HTML and CSS with multiple interactive pages simulating a digital payments platform.

-Design Features
Fully Responsive Design - Works perfectly on mobile, tablet, and desktop
PayTM Color Scheme - Uses PayTM's signature blue (#00BAF2) and clean white design
Modern UI/UX - Clean layout with intuitive navigation
Interactive Elements - Hover effects, transitions, and visual feedback
Font Awesome Icons - For enhanced visual experience

-Multi-page Navigation - 3 different fully functional pages
Digital Wallet - Wallet balance with transaction history
UPI Payments - Send/receive money with QR code

-Pages Included
Home Page (index.html) - Landing page with hero section, services, and offers
Wallet Page (wallet.html) - Digital wallet with balance and transactions
UPI Payments (upi.html) - Send/receive money using UPI
